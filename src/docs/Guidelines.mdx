# Design System Guidelines

Welcome to our design system guidelines. This document provides standards for using our components.

## Color Usage

Use our color system consistently to maintain brand identity:

- **Primary Blue** (#2563EB): Use for primary actions and key interactive elements
- **Secondary**: Use for alternative actions and supporting elements
- **Grey Scale**: Use for text, backgrounds, and dividers

## Typography

Our type system is designed for readability and hierarchy:

| Style     | Font  | Weight | Use Case        |
|-----------|-------|--------|-----------------|
| Heading 1 | Inter | 700    | Page titles     |
| Heading 2 | Inter | 600    | Section headers |
| Body      | Inter | 400    | Regular text    |

## Spacing System

Consistent spacing creates visual harmony:

- 4px = Extra small (xs)
- 8px = Small (sm)
- 16px = Medium (md)
- 24px = Large (lg)
- 32px = Extra large (xl)
